HTML

HTML, ou HyperText Markup Language, é a linguagem padrão utilizada para estruturar e apresentar conteúdo na web. Como base fundamental de qualquer página da web, HTML utiliza uma estrutura de elementos organizados hierarquicamente para definir o conteúdo, como texto, imagens, vídeos e outros tipos de mídia.

Cada elemento é representado por uma tag que especifica sua função e como ele deve ser exibido no navegador.

CSS

CSS, ou Cascading Style Sheets, é a linguagem usada para estilizar a apresentação de documentos HTML e XML. Com CSS, os desenvolvedores podem controlar a aparência e o layout de uma página da web, definindo cores, fontes, espaçamentos, alinhamentos e outros aspectos visuais.

A separação entre conteúdo (HTML) e apresentação (CSS) permite uma maior flexibilidade e manutenção de código, pois alterações no estilo de um site podem ser feitas sem modificar o conteúdo subjacente. CSS utiliza seletores para aplicar estilos a elementos específicos ou conjuntos de elementos, oferecendo uma ampla gama de técnicas de formatação que vão desde o básico até técnicas avançadas como flexbox e grid layout.

JAVASCRIPT

JavaScript é uma linguagem de programação amplamente utilizada para desenvolvimento web. Ela permite aos desenvolvedores criar interações dinâmicas em páginas da web, adicionando comportamentos e funcionalidades complexas. Originalmente desenvolvida para manipular elementos em browsers, JavaScript evoluiu significativamente ao longo dos anos e hoje é fundamental não apenas para front-end, mas também para back-end através de frameworks como Node.js. Sua sintaxe simples e flexível, combinada com uma vasta gama de bibliotecas e frameworks, torna JavaScript uma escolha popular para construir aplicações modernas e responsivas.

Exemplos de funções:

  • forEach()
  • map()
  • filter()
  • fetch()
  • split()
BOOTSTRAP

Bootstrap é um dos frameworks front-end mais populares e amplamente utilizados para desenvolvimento web responsivo e móvel. Criado pela equipe do Twitter, Bootstrap oferece uma coleção extensa de componentes pré-construídos, como botões, formulários, navegação, modais e muito mais, todos estilizados de forma consistente usando CSS baseado em classes.

Esses componentes ajudam os desenvolvedores a construir rapidamente interfaces atraentes e funcionais sem a necessidade de começar do zero, economizando tempo e esforço no processo de desenvolvimento.
